== Description ==

This Plugin show custom content to the end of each post.

For more information, check out <a href="http://www.danycode.com/end-post-content/">End Post Content</a>

Plugin Features:

- Customizable content.
- Customizable css style.

== Installation ==

1. Upload the End Post Content plugin to your blog and activate it. 
2. Customize the plugin through: Setting -> End Post Content
